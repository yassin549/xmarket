# Everything Market

The Everything Market is a platform that allows users to trade on the "reality" of any event, combining real-world data scraping, AI-based scoring, and a limit orderbook.

## Overview

This repository contains the MVP implementation of the Everything Market, including:
- **Scraper**: Fetches news and events from RSS and other sources.
- **Reality Engine**: Scores events and updates "Reality Scores" for stocks.
- **Orderbook**: A limit orderbook for trading.
- **Frontend**: A React dashboard for real-time updates.

## Getting Started

See `docs/plan.md` for the detailed build plan.

### Prerequisites
- Python 3.11+
- Node.js 18+
- Docker & Docker Compose

### Running Locally
```bash
docker-compose up --build
```
